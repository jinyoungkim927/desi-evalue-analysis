# DESI E-Value Analysis

## Critical Assessment of Dark Energy Evidence

This project analyzes DESI DR2 BAO measurements using e-values to assess the robustness of evidence for dynamic dark energy.

**Key Finding:** The apparent 3-4σ evidence is NOT robust. When properly validated on held-out data, e-values drop from 392 to 1.4.

## Key Results at a Glance

| Method | E-value | σ equiv | Status |
|--------|---------|---------|--------|
| Simple Likelihood Ratio | 392 | 3.9σ | **BIASED** |
| Uniform Mixture | 15 | 2.3σ | Prior-sensitive |
| **Data-Split Validation** | **1.4** | **0.8σ** | **VALID** |
