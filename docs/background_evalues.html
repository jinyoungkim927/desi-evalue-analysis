E-Values: Mathematical Background

What Are E-Values?

E-values are a framework for hypothesis testing developed by Shafer, Vovk, Grünwald, and Ramdas, providing an alternative to p-values.

Formal Definition

An e-value is a non-negative random variable \(E\) such that under the null hypothesis \(H_0\): \[\mathbb{E}[E \mid H_0] \leq 1\]

The Likelihood Ratio E-Value

\[E = \frac{L(\text{data} \mid H_1)}{L(\text{data} \mid H_0)} = \frac{P(\text{data} \mid H_1)}{P(\text{data} \mid H_0)}\]

Why Is This an E-Value?

Under \(H_0\):

\[\mathbb{E}\left[\frac{P(\text{data} \mid H_1)}{P(\text{data} \mid H_0)} \mid H_0\right] = \int \frac{P(x \mid H_1)}{P(x \mid H_0)} P(x \mid H_0) dx = \int P(x \mid H_1) dx = 1\]

The Overfitting Problem

Warning: If \(H_1\) is chosen after seeing the data, the likelihood ratio can be arbitrarily large even when \(H_0\) is true.

Uniform Mixture E-Values

To address overfitting, average over alternatives with uniform weights:

\[E_{\text{mix}} = \int \frac{P(\text{data} \mid \theta)}{P(\text{data} \mid H_0)} \pi(\theta) d\theta\]

Data-Split E-Values

  1. Split data into training \(D_{\text{train}}\) and test \(D_{\text{test}}\)
  2. Fit the alternative using only \(D_{\text{train}}\)
  3. Compute e-value using only \(D_{\text{test}}\)
\[E_{\text{split}} = \frac{P(D_{\text{test}} \mid \hat{H}_1(D_{\text{train}}))}{P(D_{\text{test}} \mid H_0)}\]
