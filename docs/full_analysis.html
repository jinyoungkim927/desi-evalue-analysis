Friedmann equation:

\[E(z) = \frac{H(z)}{H_0} = \sqrt{\Omega_m(1+z)^3 + \Omega_{DE}}\]

Comoving distance:

\[D_C(z) = \frac{c}{H_0} \int_0^z \frac{dz'}{E(z')}\]

3. Chi-Squared Analysis

Chi-Squared Formula

\[\chi^2 = (\mathbf{d} - \mathbf{t})^T \mathbf{C}^{-1} (\mathbf{d} - \mathbf{t})\]

Computed Values

\[\chi^2_{\Lambda\text{CDM}} = 25.44\] \[\chi^2_{w_0w_a\text{CDM}} = 13.50\] \[\Delta\chi^2 = 25.44 - 13.50 = 11.94\]

Frequentist Significance

For \(\Delta\chi^2\) with 2 extra parameters:

\[p = P(\chi^2_2 > 11.94) = 0.0026\] \[\sigma = \Phi^{-1}(1 - p/2) \approx 3.0\sigma\]

4. E-Value Computation

Method 1: Simple Likelihood Ratio

Log-likelihood:

\[\ln L = -\frac{1}{2}\chi^2 - \frac{1}{2}\ln|\mathbf{C}| - \frac{n}{2}\ln(2\pi)\]

E-value:

\[E = \frac{L(\mathbf{d} | w_0w_a)}{L(\mathbf{d} | \Lambda\text{CDM})} = \exp\left(\frac{\Delta\chi^2}{2}\right) = \exp\left(\frac{11.94}{2}\right) = 392\]
⚠️ WARNING: This E = 392 is BIASED because the w₀wₐ parameters were fitted to the same data used to compute the e-value!

Method 2: GROW Mixture E-Value

Average over a grid of alternatives:

\[E_{\text{mix}} = \frac{1}{N}\sum_{i=1}^{N} \frac{L(\mathbf{d} | w_{0,i}, w_{a,i})}{L(\mathbf{d} | \Lambda)}\]

For numerical stability (log-sum-exp):

\[\ln E_{\text{mix}} = \max_i(\ln R_i) + \ln\left(\frac{1}{N}\sum_i \exp(\ln R_i - \max_j \ln R_j)\right)\]

5. Data-Split Validation

Key Method: Split data to prevent overfitting

Split Definition

  • Training set: z < 1.0 (7 data points)
  • Test set: z ≥ 1.0 (6 data points)

Step 1: Fit on Training Data

Minimize negative log-likelihood on training set only:

\[\hat{w}_0, \hat{w}_a = \arg\min_{w_0, w_a} \left[ -\ln L_{\text{train}}(w_0, w_a) \right]\]

Result: \(\hat{w}_0 = -0.872\), \(\hat{w}_a = -0.327\)

Step 2: Evaluate on Test Data

\[E_{\text{test}} = \frac{L(\mathbf{d}_{\text{test}} | \hat{w}_0, \hat{w}_a)}{L(\mathbf{d}_{\text{test}} | \Lambda)}\]

Calculation:

\[\chi^2_{\Lambda,\text{test}} = 10.23\] \[\chi^2_{w_0w_a,\text{test}} = 9.51\] \[\ln E_{\text{test}} = \frac{10.23 - 9.51}{2} = 0.36\] \[E_{\text{test}} = e^{0.36} = 1.43\]
